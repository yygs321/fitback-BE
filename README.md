# fitback-BE
### 📜 Project Description

✔과외 학생에게 F!t, 학부모님이 궁금한 정보만 F!t하게 전달하는 피드백 사이트입니다. 

✔강사는 학생의 피드백을 작성하고, 공개 여부를 설정할 수 있습니다. 

✔학부모는 학생 피드백을 달력으로 일자별 조회할 수 있습니다.
